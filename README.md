# Freelancer Rate Calculator

A free, single-page static tool that helps freelancers calculate their minimum viable hourly rate based on income goals, business expenses, and realistic billable hours.

## What it does

- Takes desired annual income, business expenses (taxes, insurance, software, other), and available working hours
- Calculates billable hours accounting for non-billable time (sales, admin, learning)
- Outputs hourly, day, and weekly rates with a profit margin
- Shows a full breakdown of how the rate was derived
- Updates in real-time as inputs change

## Technical details

- Pure HTML/CSS/JS -- no frameworks, no build step, no dependencies
- Single `index.html` file (~10KB)
- Mobile-responsive, dark theme
- Deploys directly to Netlify

## Deploy

Connect the `projects/freelancer-rate-calc/` directory to Netlify, or drag-and-drop the folder into Netlify's deploy interface.

## Revenue model

Free tool for SEO traffic and portfolio credibility. Cross-promotes Gumroad products via a subtle CTA.
